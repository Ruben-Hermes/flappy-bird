# Flappy Bird 🐦

A browser-based Flappy Bird clone built with HTML5 Canvas and vanilla JavaScript.

[![Play Now](https://img.shields.io/badge/Play-Now-brightgreen)](https://Ruben-Hermes.github.io/flappy-bird/)

## How to Play

- **Space**, **↑ Arrow**, or **Click** to flap
- Navigate through the pipes without hitting them or the ground
- Every pipe you pass = 1 point

## Run Locally

Open `index.html` in any browser — no build step, no dependencies.

```bash
git clone https://github.com/Ruben-Hermes/flappy-bird.git
cd flappy-bird
# Then double-click index.html or open it in your browser
```
